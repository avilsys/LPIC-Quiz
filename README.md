# Linux Quiz CLI

A simple command-line quiz application to practice Linux knowledge.  
Questions are stored in a YAML file, and the quiz randomly selects a number of questions for each session.

## Features

- Randomized Linux questions
- YAML-based question bank
- Multiple-choice answers
- Score summary at the end
- Optional `--questions` argument to choose how many questions to ask (default: 20)

## Requirements

- Python 3.8+
- PyYAML
